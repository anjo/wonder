SOLine



Element Description

A SOLine represents itself in an SVG/XML image as a line.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOLine { [x1=xPosInt; ] [y1=yPosInt; ] [x2=xPosInt; ] [y2=yPosInt; ] [style=styleString; ] [transform=transformString; ] [class=aClassName; ]...};

Bindings

x1
The horizontal position of the upper left hand corner of the line w.r.t the origin. The upper left hand corner of the view.
y1
The vertical position of the upper left hand corner of the line w.r.t the origin.
x2
x coordinate of end of line.
y2
y coordinate of end of line.
style

A string that specifies the style applied to the rect.
An example would be:

style = "filter:url(#Turb2)";
class
CSS class.
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
