SOUse



Element Description

A SVG template defined by <svg>, <g>, or <symbol> may be instanciated by SOUse.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOUse { href=aURL; [style=styleString; ] [transform=transformString; ] [class=aClassName; ]... };

Bindings

href
UThe URI of the SVG element to be instanciated.
style

A string that specifies the style applied to the element.
An example would be:

style = "font-family:Times-BoldItalic; font-size:48 fill:#00FF00 stroke:#00FF00";
class
CSS class.
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
