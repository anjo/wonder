SOSvg



Element Description

SOSvg represents a SVG document or fragment.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOSvg { width=widthInt; height=heightInt; [style=styleString; ] [transform=transformString; ] [zoomAndPan=aCommandString; ]...};

Bindings

height
The height of the SVG.
width
The width of the SVG.
style

A string that specifies the style applied to the rect.
An example would be:

style = "filter:url(#Turb2)";
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
zoomAndPan
Can be disable, magnify or zoom.
