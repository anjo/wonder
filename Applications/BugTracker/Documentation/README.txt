BugTracker is a Wonder showcase app that links to about every Wonder framework illustrating features like:

- dummy data creation
- cool D2W pattern for factory
- plist based reporting
- charting reporting
- workflow and delegate patterns
- clazz pattern examples
- selenium functional testing
- tons and tons more stuff

Follow the guide below to get started with BugTracker:

http://wiki.objectstyle.org/confluence/display/WONDER/Getting+Started+with+BugTracker
