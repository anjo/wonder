How to Use
==========

1. Build/Install ERWOAdaptor framework

2. Include the framework in your app/project

3. Run your app with:
	
	 -WOAdaptor er.woadaptor.ERWOAdaptor 
	 
   OR alternatively:
   
	-WOAdaptor WONettyAdaptor

4. (Optionally) To run the adaptor via the web server, use: 

	-WODirectConnectEnabled false
