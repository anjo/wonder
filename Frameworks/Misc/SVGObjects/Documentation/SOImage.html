SOImage



Element Description

A SOImage displays an image in the SVG document. It corresponds to the SVG/XML element <image src="url">.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOImage { filename=imageFileName; | [framework=frameworkBaseName | "app" ;] | href=aURL; | width=anInt; height=anInt; [style=styleString; ] [transform=transformString; ] [class=aClassName; ]... };

Bindings

width
The horizontal dimension of the image.
height
The vertical dimension of the image.
filename
Path to the image relative to the WebServerResources directory.
href
URL containing the image data. Use this attribute for complete URLs; for relative URLs use filename instead.
framework
Framework that contains the image file. This attribute is only necessary if the image file is in a different location from the component. That is, if the component and the image file are both in the application or if the component and the image file are both in the same framework, this attribute isn't necessary. If the image file is in a framework and the component is in an application, specify the framework's name here (minus the .framework extension). If the image file should be in the application but the component is in a framework, specify the "app" keyword in place of the framework name.
style

A string that specifies the style applied to the text. This is mutually exclusive to setting one or more of the tags: fontFamily, fontSize, stroke, strokeWidth, fill.
An example would be:

style = "font-family:Times-BoldItalic; font-size:48 fill:#00FF00 stroke:#00FF00";
class
CSS class.
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
