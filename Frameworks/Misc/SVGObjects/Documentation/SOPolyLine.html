SOPolyLine



Element Description

A SOPolyLine represents a set of connected straight line segments.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOPolyLine { points=pointsString; [style=styleString; ] [transform=transformString; ] [class=aClassName; ]...};

Bindings

points
The points that make up the polyline.
style

A string that specifies the style applied to the rect.
An example would be:

style = "filter:url(#Turb2)";
class
CSS class.
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
