SOTextSpan



Element Description

A SOTextSpan represents <tspan>. It can be used to adjust text properties within a <text> element.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOTextSpan { [x=positionInt; ] [y=positionInt ; ] [style=styleString; ] [class=aClassName; ]...};

Bindings

x
Absolute or relative x coordinate of text.
y
Absolute or relative y coordinate of text.
style

A string that specifies the style applied to the text. This is mutually exclusive to setting one or more of the tags: fontFamily, fontSize, stroke, strokeWidth, fill.
An example would be:

style = "font-family:Times-BoldItalic; font-size:48 fill:#00FF00 stroke:#00FF00";
class
CSS class.
