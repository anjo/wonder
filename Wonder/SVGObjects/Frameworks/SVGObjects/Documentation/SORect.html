SORect



Element Description

A SORect represents itself in an SVG/XML page as a rectangle.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SORect { [x=xPosInt; ] [y=yPosInt; ] width=widthInt; height=heightInt; [rx=xPosInt; ] [ry=yPosInt; ] [style=styleString; ] [transform=transformString; ] [class=aClassName; ]...};

Bindings

height
The height of the rect.
width
The width of the rect.
x
The horizontal position of the upper left hand corner of the rect from the origin. The upper left hand corner of the view.
y
The vertical position of the upper left hand corner of the rect from the origin.
style

A string that specifies the style applied to the rect.
An example would be:

style = "filter:url(#Turb2)";
class
CSS class.
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
rx
The x-axis radius of the ellipse used to round off the corners of a rounded rectangle.
ry
The y-axis radius of the ellipse used to round off the corners of a rounded rectangle.
