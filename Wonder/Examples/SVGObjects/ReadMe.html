SVGObjects Examples Overview

Installation

MacOS X

To install the SVGObjects framework, open up Terminal.app and from the command line cd to the SVGObjects directory and type: 

> pbxbuild install
> su root
> cd /tmp/SVGObjects.dst/
> ditto . /

Windows 2000

In ProjectBuilderWO open the SVGObjects Framework project, choose Options from the Build Panel, select install and hit "Build".

Running the Examples

Once the SVGObjects Framework has been installed, you may build & run the example apps as normal.

The examples consist of the following projects:
EZForms Application previewing PDF-quality documents as SVG
Hello Simple demonstration of generating typographic images on the fly
ComponentTour Mostly for developers to see examples of SVGObjects components and their bindings

Hello (simple)

Programming tasks:
  • Create SVG page and components
  • Use WOSVG to embed a SVG image on a n HTML page

EZForms (medium)

Programming tasks:
  • Basic EOF integration
  • Use of stateless and non-synchronizing SVG components
  • Raster image integration using SOImage.
  • Frames in an SVG WebObjects app.

ComponentTour (simple)

Programming tasks:
  • Use of WOXMLNode to markup SVG tags for WebObjects.
  • Demonstration of SOImage, SORect and SOCircle.
  • Use of directActions in an SVG WebObjects app.
Examples by: Ravi Mendis.
