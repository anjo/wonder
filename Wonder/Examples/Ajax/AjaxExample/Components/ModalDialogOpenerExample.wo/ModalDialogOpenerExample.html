AjaxModalDialog Used in a Form

Demonstrates an AjaxModalDialog with inputs incorrectly nested in a form:
  • buttons are inactive as the dialog is not rendered in the context of a form
  • close X still works
  • warning in log: The AjaxModalDialog should not be used inside of a WOForm
    (EmployeeForm) if it contains any form inputs or buttons. Remove this AMD from this
    form, add a form of its own. Replace it with an AjaxModalDialogOpener with a dialogID
    that matches the ID of this dialog.

AjaxModalDialogOpener Used in a Form

Demonstrates an AjaxModalDialogOpener used in a form:
  • AjaxModalDialog is physically outside of the form
  • AjaxModalDialog contains its own form
  • AjaxModalDialog has an explicit ID
  • AjaxModalDialog uses showOpener = false binding
  • AjaxModalDialogOpener used inside of form to open AjaxModalDialog
  • AjaxModalDialogOpener overrides title specified on AjaxModalDialog
  • AjaxModalDialogOpener uses child components as link contents
