SOFilter



Element Description

A SOFilter represents a filter effect.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOFilter { [x=xPosInt; ] [y=yPosInt; ] [width=widthInt; ] [height=heightInt; ] [style=styleString; ] [href=sURI; ] [class=aClassName; ]...};

Bindings

href
UThe URI of another filter element within the document
height
The height of the largest possible screen buffer.
width
The width of the largest possible screen buffer.
x
The x coordinate the largest possible screen buffer.
y
The y coordinate the largest possible screen buffer.
style

A string that specifies the style applied to the rect.
An example would be:

style = "filter:url(#Turb2)";
class
CSS class.
