SOEllipse



Element Description

A SOEllipse represents an ellipse - a primitive shape.

Additional bindings prefixed with "$" get appended to the style attribute.


Synopsis

SOEllipse { [cx=inInt; ] [cy=inInt; ] rx=anInt; ry=anInt; [style=styleString; ] [transform=transformString; ] [class=aClassName; ]... };

Bindings

rx
The x-axis radius of the ellipse.
ry
The y-axis radius of the ellipse.
cx
Horizontal offset of circle from the orgin. (The upper left hand corner of the screen).
cy
Vertical offset from the orgin.
style

A string that specifies the style applied to the circle. This is mutually exclusive to setting one or more of the tags: opacity, strokeWidth, fill.
An example would be:

style = "fill:#00FF00 stroke:#00FF00";
class
CSS class.
transform
You may apply various transforms to the text: rotate, scale, translate, or matrix.
