The GoogleChart framework provides WO wrappers around Google's charting API. The WO Component API was inspired by the Ruby Googlecharts library (http://googlecharts.rubyforge.org/). GoogleChart framework depends on Ajax framework for decoding array variables, so you must include Ajax framework in your application, though you don't need to actually be an Ajax application.

The examples are taken nearly directly from the Ruby library examples page, and the framework supports the definition of data both with bindings to NSArrays of data as well as inline-String definitions of data using JSON syntax. Also, their examples page looks a lot nicer :)

See the ERComponentTour example application for examples of the use of these components.

Chart Types/Components:

  • GCBarChart
  • GCLineChart
  • GCMap
  • GCMeter
  • GCPie3DChart
  • GCPieChart
  • GCQRCode
  • GCRadarChart
