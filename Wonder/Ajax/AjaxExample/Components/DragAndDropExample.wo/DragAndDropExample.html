Droppable is a little weird, because we don't have actual objects to reference for the draggables. As a result, your droppable is notified of the droppedDraggableID (defaults to the ID of the draggable). Additionally, if you specify and action, it will call that action method when the drop is completed. The onDrop javascript function is called AFTER the action method is called.
