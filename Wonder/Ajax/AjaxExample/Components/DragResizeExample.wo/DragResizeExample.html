disable editing to see text on hover Ajax image annotations example.
You can move around the annotations, add new ones, delete selected annotations and edit the contents of the annotations.
The changes are sent to the server via AJAX. You can use the same Components to add stickies to your page.
Ajax requests have been disabled for the stickies. The javascript behaves oddly when you try to resize an element, when there is no other element below it.
Try to resize one of the stickies in their default position and you see what I mean.
When you move the stickie over the image on the left, resize will work fine.
